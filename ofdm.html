Basic idea:¶

we want to send some data

OFDM is a way to encode the data for sending

data stored as magnitude and phase in the frequency domain, and send out as wave

The transmitted wave is just: $\sum e^{i (2 \pi f + phase)}$

i.e. many sine wave of different freq, phase, magnitude sent in parallel (superpositioned)

data_tx = np.floor(np.random.random(N_sam)*2)        # data to be transmitted
print('data_tx',''.join(map(str,map(int,data_tx))))
freqs_tx = data_tx*2-1                               # put data into freq spectrum, each is called a data carrier. 
                                                     # bpsk: const magnitude, -1/1 phase

t_cplx_tx = np.fft.ifft(freqs_tx)                    # complex time domain data


########################################################################################################
# lets skip the part that transform complex time data back and forth to real signal
########################################################################################################

####################### recieve ######################
t_cplx_rx = t_cplx_tx + np.random.randn(N_sam)*.025    # recieved time domain data, has some noise
freqs_rx = np.fft.fft(t_cplx_rx)                      # recieved freq domain data
data_rx = (freqs_rx.real + 1)/2                       # recieved data

print('data_rx', ''.join(map(str,map(int,data_rx+.5))))

OFDM的原理¶

傳統FDM¶

~众所周知，~傳統fdm是在不同頻率 發送几個不同phase/amplitude 的 sine wave

為免互相干擾, 不同頻率的wave中間会留些空間, 浪費了帶寬

OFDM¶

每个 carrier 在頻域实際上是一个sinc (sin(f)/f), 在時域上就是一个很簡單的長方形

可以看到sinc是有很多零点的 (因為分母的sin)

当把几个sinc 的中心放在互相間的零点, 那中心就不会收到干擾了, 从而实現高效運用帶寬

其他¶

Pilot Carrier¶

如果信號內容全是 接收端方未知的data, 接收端就無法得知哪裡是信號的起點,

所以需要一些相方約定內容來实現同步 (可能还有其他用途)

pilot carrier 就是固定了的一些carrier, 不帶data

而帶data的carrier 就叫data carrier
